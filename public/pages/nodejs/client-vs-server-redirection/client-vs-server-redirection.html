Client vs Server redirection

Introduction

There is 2 main types og web-redirection they are called client and server redirection:
They both sends a user to a different web page, but they achieve in 2 different ways.
Client redirection is done by the browser, and server redirection is done by the server.
Here is some examples:

Client redirection

Client redirection is done by the browser, and it is done by using the location object.
The location object is a property of the window object.
The location object has a property called href, and this property is used to set the url of the page that the user will be redirected to.
Here is an example: 

                
    window.location.href = '/visitors';

Server redirection

Server redirection is done by the server, and it is done by using the res.redirect() method.
The res.redirect() method is a method of the res object.
The res.redirect() method takes a url as a parameter, and this url is the url of the page that the user will be redirected to.
Here is an example: 

                
 app.get("/api/guards", (req, res) => {
    if (req.query.passport === "theskyisblue") {
      return res.redirect("/api/tanks")
    }
    res.send({ error: "You are not allowed to enter the museum" })
  });
