Express and Nodemon

Express

Express is a Node.js web application framework, which is used to build web applications and APIs.

Installation

To install Express, open a terminal and run the following command: 

                 
        npm install express

Example

Here is a simple example with "Hello World" in Express: 

                    
    const express = require('express')
    const app = express()

    app.get('/', (req, res) => {
      res.send('Hello World!')
    })

    app.listen(3000, () => {
      console.log('Server started on port 3000')
    })

Response & Request

The request object represents the HTTP request and has properties for the request query string, parameters, body, HTTP headers, and so on.
The response object represents the HTTP response that an Express app sends when it gets an HTTP request.
Here is an example: 

                    
    app.get('/tank/:id', (req, res) => {
        const id = req.params.id
        // req.params.id is the id that is passed in the url
        res.send('The tank id is: ' + id)
        // res.send() is used to send a response to the client
        // res.send() can be used to send a string, an array, an object, a buffer, etc.
    })

Routing

Routing is how the applicationâ€™s endpoints respond to client requests.
The following code is an example of a simple route: 

                    
    app.get('/', (req, res) => {
        res.send('Hello World!')
    })

it means that when the client sends a GET request to the root path (/), the app will respond with "Hello World!".
You can also use the POST, PUT, DELETE, and other HTTP methods.
Here is an example: 

                    
    const listOfNames = []
    app.post('/addName', (req, res) => {
        listOfNames.push(req.body.name)
        res.send('Got a POST request with the name: ' + req.body.name)
    })

It means that when the client sends a POST request to the path (/addName) with the key "name" in the body,
the app will respond with "Got a POST request with the name: + (the value of the key that was sent with the body)".

Nodemon

Nodemon is a tool to help you develop node.js applications.
What nodemon does is that it watches for any changes in your source and automatically restarts your server.
This is very useful when you are developing an application and you want to see the changes you made immediately.
Also nodemon runs your application and with express you can create a server and run it.

Installation

To install Nodemon, open a terminal and run the following command: 

                    
        npm install -g nodemon

The -g flag installs nodemon globally, so you can use it from any directory.

Example

To use nodemon, we replace the node command with nodemon in the command line. 

                    
          nodemon app.js

Nodemon will start your application and monitor the directory for changes.
If it detects any changes, it will automatically restart the server.
Inside of Nodemon.json, there is a key called "ext" here you can change the value so you tell it which
files nodemon should look after so it knows when to restart when the files changes

    Nodemon.json
                        
    {
        "ext": "js, json, html"
    }
