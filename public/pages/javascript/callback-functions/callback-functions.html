Functions & Callback Functions

Introduction

We talked about variables that you can read about here.
Functions is a block of code that can be executed when called.
Functions are declared with the function keyword.
Callback functions are functions that are passed as arguments to other functions.
Callback functions are executed after the current function has finished.

Functions

Functions are declared with the function keyword.
Functions are executed when they are called.
Functions can be called as many times as you want.
Functions can be declared before or after they are called.
Functions can be declared with parameters.
Functions can be declared with a return value.

Here is some examples:

To define a function, you use the function keyword followed by the name of the function and a set of parentheses.
Inside the parentheses, you can list any arguments that the function will take.
Finally, you use curly braces to enclose the code that makes up the function's body.

    Example 1:
                    
    function myFunction() {
        console.log("Hello World!");
    }
    // Call the function
    myFunction();
    // Output: Hello World!
                    
                 
    Example 2:
                    
    function addNumbers(a, b){
        return a + b;
    }                
    // Call the function
    console.log(addNumbers(1, 2));
    // Output: 3
                    
                 
    Example 3:
                    
    function greetings(name){
        return "Greetings " + name + " my good old friend.";
    }
    // Call the function
    console.log(greetings("John"));
    // Output: Greetings John my good old friend.

Callback Functions

Callback functions are functions that are passed as arguments to other functions.
Callback functions are executed after the current function has finished.
Here is some examples: 

    Example 1:
                    
    function greetings(name, callback) {
        console.log('Hello, ' + name + '!');
        // Output: Hello, John!
        callback();
        // Call the callback function
    }
    
    function sayGoodbye() {
        console.log('Goodbye!');
    }

    greet('John', sayGoodbye);
    // Output: Hello, John!
                    
                
    Example 2:
                    
    const port = 8080;
    app.listen(port, (error) => {
        if (error) {
            // If there is an error, log it to the console
            console.log(error);
        }
        // If there is no error, we log the port number to the console
        console.log(`Server started on port ${port}`);
    });

The app.listen() function takes two arguments, the port number and a callback function.
The callback function is executed after the server has started.
The callback function takes an error as an argument.
