Variables & Data Types

Variables

Variables are kind of containers to store data values.

There are 3 ways to declare a variable in JavaScript:

                
    var x = 5;
    let y = 10;
    const z = 15;

Difference between var, let and const

var: you can declare a variable that can be changed later.
let: you can also declare a variable that can be changed later.
const: you can declare a variable that cannot be changed later.
Here is some examples: 

                
    var x = 5;
    x = 10;
    console.log(x); // 10

    let y = 5;
    y = 10;
    console.log(y); // 10

    const z = 5;
    z = 10; // error

    const obj = {
        name: "John",
        age: 30
    };
    // even if the object is a const, you can still change the properties
    obj.name = "Peter"; // This is allowed

Data Types

JavaScript has dynamic data types. This means that the same variable can be used to hold different data types:

                
    var x; // Now x is undefined
    x = 5; // Now x is a Number
    x = "John"; // Now x is a String

JavaScript has these data types:

  • String
  • Number
  • Boolean
  • Array
  • Object
  • Function
  • Undefined

Here are some examples:

                
    var x = "John"; // String
    var y = 123; // Number
    var z = true; // Boolean
    var a = ["John", "Peter", "Sally"]; // Array
    var b = {name: "John", age: 30}; // Object
    var c = function() { // Function
        console.log("Hello World!");
    };
    var d; // Undefined
