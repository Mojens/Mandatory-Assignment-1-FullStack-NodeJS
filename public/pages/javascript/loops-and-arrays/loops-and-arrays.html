Loops & Arrays

Introduction

Loops and arrays are two very important concepts in javascript.
They are used in almost every program you will ever write.

Arrays

Arrays are used to store multiple values in a single variable.
They are declared using square brackets and can be initialized with a set of elements.
Here is an example: 

                    
    const numbers = [1, 2, 3, 4, 5];
    const fruits = ["apple", "banana", "orange"];
    const persons = [
      {name: "John", age: 30},
      {name: "Mary", age: 25},
      {name: "Bob", age: 40}
    ];

Arrays are zero-indexed.
This means that the first element in an array is at index 0.
The second element is at index 1, and so on.

How to enter an array

To enter an array, you use the index of the element you want to enter.
Here is an example: 

                    
    console.log(numbers[0]); // Output: 1
    console.log(fruits[1]); // Output: "banana"
    console.log(persons[2].name); // Output: "Bob"

Loops

Loops are used to repeat a block of code a certain number of times.
There are three types of loops in javascript: for, while and do-while.
Here is some examples:

For loops

For loops are used to repeat code a certain number of times.
Here is an example: 

                    
    for (let i = 0; i < 5; i++) {
        console.log(i);
    }

This will output the numbers 0 to 4.
The first part of the for loop is where we declare the variable we'll use in the loop.
Afterwards we write the condition, where it must be true for the loop to continue in this part (i is smaller than 5).
The last part is where we increment the variable.

While loops

While loops are used to repeat code while a condition is true.
Here is an example: 

                    
  const numbers = [1, 2, 3, 4, 5];
  let i = 0;  
  while (i < numbers.length) {
    console.log(numbers[i]);
    i++;
  }

This will console.log the numbers 1 to 5.
It has The condition that as long i is smaller than the length of the array it will keep doing whats inside the codeblock in the while-loop.
The variable i is incremented in the loop. So after 5 times it would stop because then the condition is not true anymore

Do-while loops

Do-while loops are used to repeat code while a condition is true. Just like a normal while loop
Here is an example: 

                    
    let i = 0;

    do {
      console.log(i);
      i++;
    } while (i < 5);
