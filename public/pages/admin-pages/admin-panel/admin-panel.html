Admin panel

Welcome to the Admin panel! Here, you have access to a range of tools and features to help you manage your account and create new documentation pages.

Profile

At the right side of the page, you'll see a sidebar with links to different sections. Click on "Profile" to view and update your personal information, including your name, email address, and password.

New Documentation Page

When you're ready to create a new documentation page, click on "New Documentation page" to get started. You'll be able to fill out the content for a new documentation page.
