Get-request

Anatomy of an URL

Query Parameters

Query parameters are used to send additional data along with a request. In Express, you can use the request parameters using (req.query)
Here's an example:

The URL would look like this:
http://localhost:8080/bat?adjective=spooky 

            
    // GET method
    app.get("/bat", (req, res) => {
        res.send({ message: `The bat is ${req.query.adjective}` })
    });

Path Parameter

Path parameters are usually used to send a id with a request. In Express, you can use the Path parameters using (req.params)

The URL would look like this:
http://localhost:8080/api/birds/1 

            
    // GET method
    app.get('/api/birds/:id', (req, res) => {
        const bird = Birds.find(b => b.id === parseInt(req.params.id));
        // find the bird with the given ID
        if (!bird) {
            res.status(404).send(`Could not find Bird with ID: ${req.params.id}.`);
            // if the bird is not found, return a 404 error
        };
        res.send(bird);
        // if the bird is found, return the bird
    });
